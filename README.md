# Telegram Video Bot (YouTube / VK / Instagram / RuTube)

## Если BotHost не показывает логи и бот не стартует
Сделай максимально совместимый запуск:
1. В корне проекта должен быть `bot.py`, `main.py`, `app.py`, `run.py`.
2. В BotHost поставь команду запуска: `python bot.py`.
3. Если команду менять нельзя, обычно платформа пробует `main.py`/`app.py` автоматически — они тоже добавлены.
4. Убедись, что в корне есть `token.txt` с токеном одной строкой.

Пример `token.txt`:
```text
123456:ABCDEF...
```

## Быстрый запуск для BotHost (без env)
1. Создай `token.txt` рядом с `bot.py`.
2. Перезалей проект полностью.
3. Нажми `Clean rebuild`.
4. Оставь только 1 инстанс.

## Основной функционал
- ссылки YouTube/VK/Instagram/RuTube
- выбор качества
- очередь загрузок
- лимиты размера/длительности
- `reply_video` для mp4, иначе `reply_document`
- очистка временных файлов
